# 臨床工学技士国家試験 過去問演習サイト

ユーザー添付の第39回臨床工学技士国家試験（午前・午後）PDFと解答番号PDFをもとに作成した、静的Webアプリです。

## 主な機能

- 午前・午後に分けた問題演習
- 5択ボタン形式での解答
- 正答率・経過時間の表示
- 正答表に基づく判定表示
- 図表問題では原本ページ画像を表示
- 間違えた問題の復習
- ランダム演習
- ブックマーク
- 解答履歴の保存
- ダークモード

## 収録データ

- 第39回 臨床工学技士国家試験 午前：90問
- 第39回 臨床工学技士国家試験 午後：90問
